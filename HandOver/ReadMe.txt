     ReadMe.txt created by Yeojung

     1. 해당 폴더에 있는 파일들은 인수인계를 위해 새 폴더(NVspin_control)에 정리된 자료입니다.
          또한 폴더에 순서가 유의미한 경우는 001, 011과 같이 표시해두었으니 참고 바랍니다.
     
     2. 기존 폴더의 경로 위치가 남아있으므로 바로 코드를 실행한다면 오류가 생성될 수 있습니다.
          그러므로 폴더에서 바로 실행하지 마시고 자신의 개인 폴더에 복사하여 사용하시기 바랍니다.
     
     3. 모든 파일마다 코드 실행 전 '파일 읽기', '파일 쓰기', '폴더 생성' 등의 과정이 있는지 확인하시기 바랍니다.
          해당 과정이 존재한다면 자신의 환경(로컬 or 서버 등)에 맞추어 경로를 수정하시면 오류 가능성이 줄어듭니다.

     4. .ipynb 파일에는 기존에 실행했던 기록이 필요에 따라 참고할만한 자료로 남아있습니다.
          그러므로 폴더에서 바로 실행하지 마시고 자신의 개인 폴더에 복사하여 사용하시기 바랍니다.

     5. 시간 부족 이슈로 인해 코드마다 미완성 부분이 있을 수 있습니다.
          -> 설명이 부족한 경우 or 영어/한글이 혼용된 경우 or 코드(ex. 함수) 중복 사용 or 코드 오류 고침 실패 등 

     6. 코드나 폴더에 관한 설명은 NVspin_control/HandOver_comments.txt 파일에 기재되어 있습니다.

     7. 프로젝트 진행 상황
          - A* path finder : 기존 A* 코드에서 균일화 적용 위해 haar measure 적용(theta, phi에 omega 추가)
          - Deep Learning(ML) : 
               > 주어진 문제 해결을 위해 RNN 모델에서 simpleRNN, LSTM, GRU 비교로 GRU 결정
                    1) theta와 phi를 입력으로 sequence 출력하는 모델(성공), 
                    2) rho(matrix)를 입력으로 sequence 출력하는 모델(복소수 이슈로 실패),
                    3) theta_x/y/z를 입력으로 sequence 출력하는 모델(최종) 생성하고 효용 확인 완료
               > 튜닝 시도했으나 시간 부족으로 인해 미완성(모델 개선 가능성 남아있음)
               > 주로 사용했던 데이터셋은 omega값 없는 샘플 31595개 데이터(ByAstar_dt_2.6_1130.csv)
                    HandOver/NVspin_control/001_Path_Finder_AStar/012_By_Yeojung/Data/
                    Non-Haar_measure_None_Omega/using/dt_2.6/ByAstar_dt_2.6_1130.csv
          - Local Optimizer : brute-force하고 검증이 덜 되었지만 샘플 개당 길어도 1~2분 내에 끝나는
                              단순 local optimizer 완성(BFGS 적용 버전을 시도했으나 여러 이슈로 실패)
          - 남은 이슈 : A* 결과 데이터에서 생기는 결측 구간으로 인해 비교적 정확도 떨어지는 모델 생성됨
                         ... d0 값 수정하며 결측 구간 완화를 통해 약간의 개선점 확인(추가 조사 필요)
